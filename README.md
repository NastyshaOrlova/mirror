# Mirror

Модульное приложение для организации ежедневных ритуалов. Набор модулей не фиксирован — инструменты появляются и убираются в зависимости от текущих целей.

> 🚧 Проект в активной разработке!

## Текущие модули

- **Auth** — регистрация, JWT-аутентификация
- **Notes** — дневник
- **Morning Reminder** — утреннее напоминание
- **Evening Question** — вечерняя рефлексия
- **Checklist** — трекинг задач
- **Export** — выгрузка данных за период

## Стек

- Node.js, NestJS, TypeScript
- PostgreSQL, TypeORM

Клиентская платформа (мобильное / веб / десктоп) определяется после стабилизации API.

## Запуск

```bash
# зависимости
npm install

# dev-режим
npm run dev
```
